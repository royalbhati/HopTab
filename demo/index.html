And More

Built for power users

⚙

Pin from Anywhere

Pin from running apps or browse all installed apps. Pin apps before you even open them.

⌨

Multi-Monitor

Overlays follow your cursor to the active screen. Layouts apply to the correct monitor.

✨

App Actions

Cmd+Q, Cmd+H, Cmd+M while the switcher is open to quit, hide, or minimize any app.

⇄

Click to Switch

Click any app icon in the overlay to switch directly. No keyboard required.

🛠

Auto-Unminimize

Minimized apps are automatically restored when activated through HopTab. No more fishing in the dock.

⎕

Menu Bar Native

Lives quietly in your menu bar. No Dock clutter, no distractions. Just there when you need it.

Get Started

Install in seconds

# Download, unzip, and clear quarantine
curl -sL "$(curl -s https://api.github.com/repos/
  royalbhati/HopTab/releases/latest \
  | grep -o '"browser_download_url": *"[^"]*"' \
  | head -1 | cut -d '"' -f 4)" \
  -o /tmp/HopTab.zip \
  && unzip -o /tmp/HopTab.zip -d /Applications \
  && xattr -d com.apple.quarantine \
  /Applications/HopTab.app
1
Run the command or download from Releases
2
Grant Accessibility permission when prompted
3
Pin your apps and press ⌥+Tab
