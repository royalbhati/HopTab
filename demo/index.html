Features

Everything you need,
nothing you don't

⚙

Pin Any App

Pick from running apps in Settings. Click to pin or unpin. Your selections persist across restarts.

⌨

Global Hotkey

Option+Tab by default. Also supports Control+Tab or Option+Backtick. Works everywhere, always.

✨

Native Overlay

Beautiful vibrancy blur panel with app icons. Non-activating panel floats above everything without stealing focus.

⇄

Cycle Both Ways

Tab to go forward, Shift+Tab to go backward. Release the modifier to activate. Escape to cancel.

🛠

Profiles

Create named profiles for different workflows. "Coding", "Design", "Writing" — each with their own pinned apps.

⎕

Desktop Assignment

Bind profiles to macOS Spaces. Swipe to Desktop 2 and your pinned apps automatically change.

Get Started

Install in seconds

# Download, unzip, and clear quarantine
curl -sL "$(curl -s https://api.github.com/repos/
  royalbhati/HopTab/releases/latest \
  | grep -o '"browser_download_url": *"[^"]*"' \
  | head -1 | cut -d '"' -f 4)" \
  -o /tmp/HopTab.zip \
  && unzip -o /tmp/HopTab.zip -d /Applications \
  && xattr -d com.apple.quarantine \
  /Applications/HopTab.app
1
Run the command or download from Releases
2
Grant Accessibility permission when prompted on first launch
3
Pin your apps and press ⌥+⇥ to start hopping
